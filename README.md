# Wine Classification - Support Vector Machine (SVM)

## 📌 Contexte du Projet
Ce projet de Classification Multiclasse a pour but d'identifier l'origine de trois types de vins italiens à partir de leur composition chimique. Projet réalisé en Master 1 SAAD.

## 🛠️ Stack Technique
* **Algorithme :** Support Vector Machine (SVM) avec noyau linéaire.
* **Prétraitement :** Normalisation via `StandardScaler`.
* **Évaluation :** Matrice de confusion et Classification Report (F1-Score).

## 📊 Résultats Principaux
* **Précision globale : 97%**.
* **Fait marquant :** Les caractéristiques chimiques comme la **Proline** et l'**Alcool** sont les signatures dominantes qui permettent de différencier les trois vignobles.
* Une seule erreur de classification sur l'ensemble de test, démontrant la robustesse de l'approche SVM sur des données standardisées.
